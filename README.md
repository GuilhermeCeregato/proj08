# 📚 Atividade - Estrutura de Dados em JavaScript
## 📖 Sobre a atividade
Esta atividade foi desenvolvida com o objetivo de praticar conceitos de **Estrutura de Dados** utilizando JavaScript, aplicando:

- Vetores
- Matrizes
- Manipulação de arrays
- Métodos nativos do JavaScript

---
## 🚀 Conteúdos utilizados

Durante a atividade foram utilizados os seguintes conceitos:

### 🔹 Vetores
Criação e manipulação de arrays contendo objetos com informações dos alunos.

### 🔹 Matrizes
Utilização de matrizes para armazenamento de notas dos alunos.

### 🔹 Métodos de Arrays
Foram aplicados diversos métodos importantes do JavaScript:

| Método | Função |
|---|---|
| `splice()` | Remove elementos do vetor |
| `forEach()` | Percorre os elementos |
| `map()` | Cria novos vetores |
| `filter()` | Filtra informações específicas |
| `reduce()` | Realiza cálculos e comparações |
